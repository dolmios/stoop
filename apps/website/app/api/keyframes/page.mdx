# keyframes

<Icon name="Play" size={24} style={{ display: "inline", verticalAlign: "middle", marginRight: "8px" }} /> Create CSS keyframe animations with theme tokens

## Usage

```typescript
import { keyframes, css } from "./theme";

const fadeIn = keyframes({
  from: {
    opacity: 0,
  },
  to: {
    opacity: 1,
  },
});

const animatedClass = css({
  animation: `${fadeIn} 0.5s ease-out`,
});
```

## With Theme Tokens

Theme tokens work directly in keyframes:

```typescript
const slideUp = keyframes({
  from: {
    opacity: 0,
    transform: "translateY($medium)",
    backgroundColor: "$primary",
  },
  to: {
    opacity: 1,
    transform: "translateY(0)",
    backgroundColor: "$secondary",
  },
});
```

Tokens are automatically resolved to CSS variables, just like in regular styles.
