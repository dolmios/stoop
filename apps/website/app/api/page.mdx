# API Reference

Complete API reference for Stoop

## Core Functions

- **[createStoop](/api/create-stoop)** - Create a Stoop instance with theme configuration
- **[styled](/api/styled)** - Create styled components with type-safe props and variants
- **[css](/api/css)** - Create CSS classes with theme tokens
- **[globalCss](/api/global-css)** - Apply global styles using theme tokens
- **[keyframes](/api/keyframes)** - Create CSS keyframe animations

## Theme Management

- **[createTheme](/api/create-theme)** - Create additional themes dynamically
- **[Provider](/api/provider)** - React component for theme switching
- **[useTheme](/api/use-theme)** - Hook to access current theme and toggle themes

## Server-Side Rendering

- **[getCssText](/api/get-css-text)** - Get all generated CSS text for SSR
- **[preloadTheme](/api/preload-theme)** - Preload theme CSS variables to prevent FOUC
- **[warmCache](/api/warm-cache)** - Pre-compile CSS objects to warm the cache
