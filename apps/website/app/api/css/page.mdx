# css

<Icon name="FileCode" size={24} style={{ display: "inline", verticalAlign: "middle", marginRight: "8px" }} /> Create CSS classes with theme tokens

## Usage

```typescript
import { css } from "./theme";

const buttonClass = css({
  padding: "$medium $large",
  backgroundColor: "$primary",
  color: "$background",
  "&:hover": {
    opacity: 0.9,
  },
});
```

## With Media Queries

```typescript
const responsiveClass = css({
  padding: "$small",
  mobile: {
    padding: "$medium",
  },
  desktop: {
    padding: "$large",
  },
});
```

## Dynamic Classes

```typescript
const dynamicClass = css({
  backgroundColor: "$background",
  color: "$text",
});
```
