# Theme Token Scales

All available theme scales in Stoop

## Approved Scales

Stoop supports 13 approved theme scales:

### colors
Color values (strings)
```typescript
colors: {
  primary: "#0070f3",
  background: "#ffffff",
  text: "#000000",
}
```

### opacities
Opacity values (strings or numbers)
```typescript
opacities: {
  disabled: "0.5",
  hover: "0.8",
}
```

### space
Spacing values (strings)
```typescript
space: {
  small: "8px",
  medium: "16px",
  large: "24px",
}
```

### radii
Border radius values (strings)
```typescript
radii: {
  small: "4px",
  default: "6px",
  large: "8px",
}
```

### sizes
Size values (strings)
```typescript
sizes: {
  small: "100px",
  medium: "200px",
  large: "300px",
}
```

### fonts
Font family values (strings)
```typescript
fonts: {
  body: "system-ui, sans-serif",
  heading: "Georgia, serif",
}
```

### fontWeights
Font weight values (strings or numbers)
```typescript
fontWeights: {
  normal: "400",
  bold: "500",
}
```

### fontSizes
Font size values (strings)
```typescript
fontSizes: {
  small: "14px",
  medium: "16px",
  large: "20px",
}
```

### lineHeights
Line height values (strings or numbers)
```typescript
lineHeights: {
  normal: "1.5",
  tight: "1.2",
}
```

### letterSpacings
Letter spacing values (strings)
```typescript
letterSpacings: {
  normal: "0",
  wide: "0.1em",
}
```

### shadows
Shadow values (strings)
```typescript
shadows: {
  subtle: "0 1px 2px rgba(0,0,0,0.05)",
  medium: "0 4px 6px rgba(0,0,0,0.1)",
}
```

### zIndices
Z-index values (strings or numbers)
```typescript
zIndices: {
  dropdown: "1000",
  modal: "2000",
}
```

### transitions
Transition/animation values (strings)
```typescript
transitions: {
  fast: "all 0.1s ease",
  default: "all 0.2s ease",
}
```
