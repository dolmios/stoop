# Installation

<Icon name="Info" size={24} style={{ display: "inline", verticalAlign: "middle", marginRight: "8px" }} /> Install Stoop using your preferred package manager

## Package Managers

<Icon name="Package" size={20} style={{ display: "inline", verticalAlign: "middle", marginRight: "6px" }} /> Stoop is available on npm and installs with any package manager:

```bash
npm install stoop
# or
bun add stoop
# or
yarn add stoop
# or
pnpm add stoop
```

## Optional: UI Component Library

<Icon name="Layers" size={20} style={{ display: "inline", verticalAlign: "middle", marginRight: "6px" }} /> Stoop also provides an optional UI component library with pre-built, accessible components:

```bash
npm install stoop-ui
# or
bun add stoop-ui
# or
yarn add stoop-ui
# or
pnpm add stoop-ui
```

The `stoop-ui` package includes components like Button, Input, Modal, Select, and more, all built with Stoop and ready to use. It requires the core `stoop` package (installed automatically as a peer dependency).

## Requirements

<Icon name="CheckCircle" size={20} style={{ display: "inline", verticalAlign: "middle", marginRight: "6px" }} /> Stoop requires React 16.8.0 or higher (for hooks support). It works with both Next.js Pages Router and App Router.
