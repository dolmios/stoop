# Installation

<Icon name="Download" size={24} style={{ display: "inline", verticalAlign: "middle", marginRight: "8px" }} /> Install Stoop using your preferred package manager

## Package Managers

<Icon name="Package" size={20} style={{ display: "inline", verticalAlign: "middle", marginRight: "6px" }} /> Stoop is available on npm and can be installed with any package manager:

```bash
npm install stoop
# or
bun add stoop
# or
yarn add stoop
# or
pnpm add stoop
```

## Requirements

<Icon name="CheckCircle" size={20} style={{ display: "inline", verticalAlign: "middle", marginRight: "6px" }} /> Stoop requires React 16.8.0 or higher (for hooks support). It works with both Next.js Pages Router and App Router.
