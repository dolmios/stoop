# Installation

<Icon name="Info" size={24} style={{ display: "inline", verticalAlign: "middle", marginRight: "8px" }} /> Install Stoop using your preferred package manager

## Core Package

<Icon name="Package" size={20} style={{ display: "inline", verticalAlign: "middle", marginRight: "6px" }} /> The build-time CSS-in-JS engine, powered by an SWC plugin:

```bash
npm install stoop
# or
bun add stoop
# or
yarn add stoop
# or
pnpm add stoop
```

## Optional: UI Component Library

<Icon name="Layers" size={20} style={{ display: "inline", verticalAlign: "middle", marginRight: "6px" }} /> Pre-built, accessible components built with stoop:

```bash
npm install stoop-ui
# or
bun add stoop-ui
# or
yarn add stoop-ui
# or
pnpm add stoop-ui
```

The `stoop-ui` package includes components like Button, Input, Modal, Select, and more, all built with stoop and ready to use. It requires the core `stoop` package (installed automatically as a peer dependency).

## Requirements

<Icon name="CheckCircle" size={20} style={{ display: "inline", verticalAlign: "middle", marginRight: "6px" }} /> stoop requires React 18 or higher. It works with both Next.js Pages Router and App Router.
